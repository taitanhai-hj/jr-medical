# Medical Management System And Admin Panel, Spring Boot, Ionic, MySQL, Hibernate, Liquibase

### Build

#### 1) サーバーサイトを起動する
   
```
$ cd server-product-management
$ gradlew bootJar
$ gradlew bootRun
```

#### 2) モバイルサイトを起動する

```
$ cd mobile-side
$ ionic serve
```

### サーバサイトにアクセスするために、下記のURLを使用する

```
http://localhost:8080
```

### モバイルサイトにアクセスするために、下記のURLを使用する

```
http://localhost:8100
```
